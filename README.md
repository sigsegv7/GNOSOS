# The GNOSOS operating system

The GNOSOS operating system aims to be a non-POSIX operating system with a hint of
inspiration from UNIX along with an opinionated design. It is a clean-slate research
project aimed to experiment with potential improvements to POSIX without dragging in
legacy requirements.

## Building

To build gnosos, simply run the ``bootstrap.sh`` script in ``host/`` before running ``make``.
